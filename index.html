Summary:

For this website, I used Figma and created a design which can be viewed on both desktop and mobile. What I wanted to achieve when making this website was a creative outlet for my projects and to be able to display them in an accessible way for a lot of people. I enjoyed this HTML and CSS project as it allowed for me to be able to understand the fundamentals in creating a website. It did take me a few go's to understand how everything links together and it took me a lot of trail and error to get my website to look and run how I wanted it to. I used parts of my Figma Web Design to create my final Webpage but decided not to go with multiple pages to avoid any confusion and a possible non-working site. Because of how new I am to this type of coding style, I felt that it was best to simplify my designs to achieve the expected criteria without losing the quality of work.
